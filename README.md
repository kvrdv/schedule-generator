# schedule-generator

Генератор рабочего расписания:

1. Программа запрашивает год и месяц
2. На основании этих данных генерируется расписание на месяц (с 1го числа) выхода на работу сотрудника, который работает сутки через двое. 
Если рабочий день, выпадает на выходные (субботу или воскресенье), то он переносится на понедельник и потом опять два выходных, один рабочий и т.д.
